💱 Currency Converter

A responsive web application that allows users to convert amounts between different currencies using real-time exchange rates. The application fetches the latest currency data from an exchange rate API and provides accurate conversions instantly through an intuitive user interface.

🚀 Features
Real-time currency conversion
Support for multiple international currencies
Dynamic exchange rate updates using API integration
User-friendly and responsive interface
Swap currencies with a single click
Input validation and error handling
Fast and lightweight design
🛠️ Technologies Used
HTML5 – Structure of the application
CSS3 – Styling and responsive design
JavaScript (ES6) – Application logic and API handling
Exchange Rate API – Real-time currency exchange rates
Fetch API – Retrieving currency data from external services
📌 How It Works
User enters an amount.
Selects source and target currencies.
Application sends a request to the exchange rate API.
Latest exchange rate is retrieved.
Converted amount is displayed instantly.
